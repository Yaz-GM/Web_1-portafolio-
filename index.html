Misión de la carrera

Formar profesionales comprometidos con la excelencia, el desarrollo tecnológico y la solución de problemas computacionales.

Visión de la carrera

  • Egresados con altos estándares y competencias técnicas.
  • Reconocimiento laboral nacional e internacional.
  • Certificaciones, dominio del inglés y experiencia práctica.

Objetivo de la materia

Capacitar al alumno en el uso de HTML, CSS y JavaScript para crear, publicar y mantener sitios web profesionales.

Consideraciones de clase

  1. Sesiones de lunes a viernes.
  2. Asistencia mínima del 90%.
  3. Material disponible en Blackboard y Teams.
  4. Tareas entregadas a tiempo y por el medio establecido.
  5. No se aceptan tareas por correo electrónico.

Contenidos de la materia

  1. HTML
  2. HTML Avanzado
  3. CSS
  4. JavaScript
  5. Diseño responsivo
